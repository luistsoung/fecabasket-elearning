# 🏀 FECABASKET E-Learning

**Plateforme officielle de formation en ligne des entraîneurs camerounais de basketball**

🌐 **Site en ligne** : [https://elearning.fecabasket.com](https://elearning.fecabasket.com) *(en cours de configuration)*
🌐 **URL Netlify** : [https://unique-hummingbird-8257b6.netlify.app](https://unique-hummingbird-8257b6.netlify.app)

---

## 🎯 À propos

Plateforme bilingue **FR / EN** développée par la **Direction Technique Nationale FECABASKET** sous l'autorité de **Louis Tsoungui**, Instructeur FIBA WABC.

- **5 niveaux de certification** alignés FIBA WABC : Animateur · Moniteur · Niveau 1 · Niveau 2 · Niveau 3
- **30 pages HTML** (15 FR + 15 EN)
- **9 équipes nationales** Lions Indomptables présentées avec photos officielles HD
- **Diapos officielles** : Window 3 Qatar 2027 World Cup Qualifiers + FIBA U17 World Cup Türkiye 2026
- **100% gratuit** pour tous les entraîneurs camerounais
- **Mobile-first**, optimisé pour les réseaux camerounais (Orange, MTN)

## 🚀 Mise à jour automatique du site (workflow)

```
Modifier les fichiers en local
        ↓
Double-clic sur MAJ-SITE.bat (bureau)
        ↓
GitHub reçoit les modifs
        ↓
Netlify détecte automatiquement
        ↓
Site mis à jour en 2 minutes (URL inchangée)
```

## 🛠️ Stack technique

- **HTML5 / CSS3 / JavaScript vanilla** (zéro dépendance, performances maximales)
- **Hébergement** : [Netlify](https://www.netlify.com/) — CDN mondial, déploiement automatique depuis GitHub
- **Polices** : Inter (Google Fonts)
- **Bilingue FR/EN** : sélecteur statique sans Google Translate (zéro latence)
- **Responsive** : mobile-first, optimisé pour les terminaux modestes
